## Tool Categories

| Toolset | Tools | Backend |
|---------|-------|---------|
| Core | terminal, file read/write/search/patch, web search/extract | Local |
| Browser | navigate, click, type, snapshot, vision, scroll | Browserbase cloud |
| Delegation | delegate_task, execute_code | Subprocess / sandbox |
| MCP | native-mcp client, mcporter bridge | External servers |
| Agent | todo, memory, cron, skill_manage | In-process |
