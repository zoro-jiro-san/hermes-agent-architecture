## Key Insights

1. **The "context rot" problem is real**: Anthropic's own research shows longer contexts degrade performance even before hitting limits. This validates Hermes's existing compression approach but suggests we need smarter compression (anchored > simple).

2. **Forgetting is a feature**: Every top memory system now implements structured decay. Hermes's flat memory store (add-only) is an outlier — we need pruning.

3. **Agent self-curation is the trend**: The best systems (CrewAI, ByteRover, Letta) have agents that actively manage their own memory. Hermes already does this partially (skill auto-patching), but the memory store lacks self-management.

4. **Full-context baselines underperform**: At 60.2% on LongMemEval, just stuffing everything in context is worse than selective retrieval. This validates Hermes's approach of injecting only profile + memory + skills.

5. **Hybrid retrieval > pure vector search**: Hindsight's 91.4% comes from combining 4 strategies. Hermes's FTS5-only approach is a significant gap.
